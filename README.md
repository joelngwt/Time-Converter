# Time Converter
An Android app to convert time between different time zones

Time zones supported:
- PST (UTC -8)
- PDT (UTC -7)
- EST (UTC -5)
- EDT (UTC -4)
- CET (UTC +1)
- CEST (UTC +2)
- SGT (UTC +8)
